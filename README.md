# 💄 L’Oréal AI Skincare Routine Generator

An AI-powered chatbot that generates personalized skincare routines based on user inputs like skin type, concerns, and preferences.

🔗 Live Demo: https://nidhilalwani01.github.io/08-prj-loreal-chatbot/

---

## ✨ Overview

This project explores how AI can be used to translate user inputs into structured, personalized recommendations—similar to how brands like L’Oréal guide product discovery at scale.

The goal was to design an experience that feels intuitive, trustworthy, and tailored, while also building the underlying decision logic that powers the recommendations.

---

## 🎯 Problem

Consumers often struggle with:
- Information overload when choosing skincare products  
- Lack of personalized recommendations  
- Confusion around routines and product combinations  

This leads to low confidence in purchase decisions and poor product-market alignment.

---

## 💡 Solution

I designed and developed an AI-powered chatbot that:
- Collects user inputs (skin type, concerns, preferences)
- Processes them through structured decision logic
- Generates a personalized skincare routine
- Recommends relevant product types in a clear, digestible format  

---

## 🧠 My Approach

This project goes beyond UI—it focuses on **how user data translates into decisions**.

- Built input → logic → output flow
- Designed decision trees for recommendation logic
- Focused on clarity and trust in AI-generated responses
- Structured outputs to mimic real-world product recommendation systems
